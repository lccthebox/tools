🌐 실시간 트렌드 크롤러 미설정

Cloudflare Worker URL 연결 시 네이버·유튜브·구글 트렌드를 실시간으로 가져와 토픽 생성에 사용해요. 미연결 시 아래 Fallback 소스풀을 사용합니다.

💡 Worker 배포: workers.cloudflare.com → Create Worker → thebox-worker.js 붙여넣기 → Deploy (무료)

Fallback 소스풀 0

🟡 크롤러 미연결 — 이 소스풀에서 랜덤 3개 사용 중이에요.

크롤러 연결 시 자동으로 실시간 트렌드가 우선 사용되고, 이 목록은 backup으로만 작동합니다.

💡 크롤러 없을 때의 보험이에요. 크롤러 연결하면 네이버·유튜브·구글트렌드가 자동으로 들어와요.

☁️ GitHub Gist 동기화 미연결

GitHub Gist에 토픽을 저장하면 어느 컴퓨터에서든 동일한 데이터를 볼 수 있어요. A에서 생성 → B에서 바로 확인 가능.

💡 토큰 발급: github.com/settings/tokens → Generate new token → gist 체크 → 복사
💡 처음엔 "지금 저장"을 누르면 새 Gist가 자동 생성되고 ID가 저장돼요.
💡 다른 컴퓨터에서는 같은 토큰 + Gist ID 입력 후 "불러오기"를 누르세요.

🗂 Google Drive 공유

토픽 뷰어를 Google Drive에 업로드해서 핸드폰으로 바로 열 수 있는 링크를 만들어요.

💡 토큰 발급: OAuth Playground → Drive API v3 scope 선택 → Authorize → Exchange → Access token 복사 (1시간 유효)

Anthropic API 키

Claude API 키를 입력하면 토픽 자동 생성이 활성화돼요. 키는 이 브라우저에만 저장됩니다.

💡 키 발급: console.anthropic.com → API Keys → Create Key

⚙️ 데이터 관리

기존에 생성된 토픽들을 새로운 빙고판/주석 폼에 맞게 최적화합니다. (내용은 유지하면서 구조만 변경)

💡 새로 바뀐 빙고판 디자인이 기존 토픽에서도 잘 보이게 설정합니다. 해석이 비어있는 칸은 직접 편집하거나 '재생성'을 이용하세요.

🔑 Anthropic API 키 입력

Claude API로 매일 새로운 토픽을 자동 생성합니다.
키는 이 브라우저에만 저장되고 외부로 전송되지 않아요.

키 발급: console.anthropic.com → API Keys → Create Key
