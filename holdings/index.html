크레딧 차감 — 더박스1 시트
출석관리 탭에서 해당 요일 차감 실행
▼
  • 더박스1 (원어민회화스터디 시트)를 띄웁니다.
  • 시트 위쪽 출석관리 탭을 눌러 차감할 요일을 선택합니다.
  • 예시: 월 수업 차감 선택 시 참여요일에 '월'이 적힌 분들이 전부 차감됩니다.
  • Ctrl + Z / 실행취소로 제대로 차감됐는지 반드시 확인하기
2
카페에서 홀딩 승인
네이버 더박스 카페 → 댓글 확인 → 크레딧 +1
▼
  • 네이버 더박스 카페로 이동합니다. → cafe.naver.com/theboxis
  • lccthebox4 / lccthebox5 / whatistheobx 중 하나로 로그인합니다.
  • 홀딩 확인할 요일의 게시판으로 들어가 해당 월의 게시글 클릭
    ex) [ 2026년 3월 ] 월수반 더박스 원어민 영어회화 스터디 홀딩 신청
  • 새댓글 확인하여 답댓글로 OO님 O회 홀딩 완료되었습니다. 라고 확인해 줍니다.
  • 답댓글로 확인한 분의 크레딧은 반드시 구글시트에서 크레딧 +1 올려주기
3
홀딩 인원 정리 — 홀딩체크 탭
요일/시간 선택 → 멤버 입력 → 홀딩/NEW/복귀 분류
▼
  • 📌 홀딩체크 탭으로 들어가 확인하려는 요일과 시간을 클릭합니다.
  • 구글 시트의 각 그룹별 인원 모두를 복사하여 한 줄로 붙여넣으면 → 한국인 멤버에 자동으로 들어갑니다.
  • 한국어 선생님의 경우 이름에서 오른쪽 클릭 → 쌤으로 이동하여 구분합니다.
  • 홀딩 게시글에서 오늘 일자 검색
    ex) 3/15일 경우 Ctrl+F로 15 검색 후 15일에 홀딩한 멤버 확인
  • 홀딩한 멤버 이름을 홀딩체크 페이지에서 검색하여 오른쪽 클릭 → 홀딩으로 이동으로 분류합니다.
  • NEW와 복귀도 오른쪽 클릭 메뉴에서 표시할 수 있습니다.
4
데일리 리포트 특이사항 확인
스케줄 변경사항 확인 → 홀딩체크 탭에 반영
▼
  • 데일리 리포트의 스터디 스케줄 변경사항 란을 확인합니다.
  • 다른 요일 참여자분이 오늘 참여한다고 되어 있을 시 → 레벨과 각 반 참여인원을 고려하여 홀딩체크 페이지에서 추가하려는 곳에 멤버 추가합니다.
5
반 편성 — 원어민 분배
홀딩 데이터 불러오기 → 원어민 추가 → 자동배치
▼
  • 반 편성 탭의 [반 불러오기]에서 요일과 시간대 선택 후 홀딩 데이터 가져오기 클릭
    예) 월수 오전 → 요일 '월' 클릭, 시간대 '오전' 클릭 → 홀딩 데이터 가져오기
  • [원어민·비원어민 추가]에서 원어민 클릭 후 현우쌤이 적어준 원어민 목록 복사(쉼표 그대로)하여 추가 / 비원어민도 동일하게 추가합니다.
  • 원어민 or 비원어민이 리더인 경우, 또는 고정 그룹을 원하는 경우 → 자동배치 전에 해당 그룹에 선편성해 두세요.
  • 이후 자동배치를 누르면 원어민·비원어민 비율이 자동으로 배치됩니다.
  • 테이블 배치는 인원수 보고 큰 그룹부터 체크 — 체크된 테이블은 중복 체크 안 됨
6
반별 출석부 · 게시판 인쇄
오른쪽 위 인쇄버튼 사용 (Ctrl+P 아님)
▼
  • 반드시 왼쪽 위 인쇄버튼을 사용합니다. Ctrl+P 사용 금지!
  • 출석부 인쇄 설정: 용지크기 A4 / 여백: 기본 / 옵션: 머릿글·바닥글 끄기, 배경 그래픽 켜기
  • 게시판 인쇄 설정: 용지크기 A3 / 여백: 기본 / 옵션: 머릿글·바닥글 끄기, 배경 그래픽 켜기
7
토픽 인쇄
전체인원 + 리더인원 수만큼 → 구글드라이브에서 PDF 인쇄
▼
  • 반 편성에서 보이는 전체인원 확인 → 전체인원 + 리더인원 수만큼 토픽을 뽑습니다.
  • 구글 드라이브 경로:
    월별 토픽 → 2026년 토픽 → 해당 월 → 스케줄 → 스케줄 뒷면(오늘의 토픽 확인) → 뒤로가기 → 토픽 → 오늘의 토픽 PDF 찾기
  • 각 반별로 해당 테이블에 나눠 놓습니다.
8
만료 크레딧 확인 및 카카오톡 안내
초록/주황 버튼 실행 → 만료자에게 카톡 발송
▼
  • 더박스1 (원어민회화 시트)로 돌아와 시트 위쪽 버튼을 눌러줍니다.
  • 🟢 초록색 버튼: 만료기간 확인 버튼
  • 🟠 주황색 버튼: 오늘 크레딧 끝난 사람 체크 버튼
  • 크레딧 만료가 된 분들이 있으면 마지막 크레딧 안내 카카오톡을 보냅니다.
  • 더박스 도구 모음 → 스터디 연장 및 크레딧 → 스터디 마지막 안내 선택 → 결과 복사 → 만료되신 분께 붙여서 보내기
  • 홀딩 텀이 길 경우, 그 사이 기간에 크레딧이 만료될 수 있으니 꼭 확인!
